# vue 的原生指令

在vue中所有的指令都是通过v- xxx
1、v-bind 等同于：
2、v-text // 直接的绑定该文本数据
3、v-html //将html标签直接解析在渲染 4、v-show // 控制显示隐藏 类似于css 的display:none
5、v-if // 动态去渲染 若不满足条件不会出现在demo 节点中
6、v-else v-else-if 就近原则
7、v-for // v-for="(item,index) in arr" :key="item" 唯一指定的key 可以减少性能，在检测有变化时，只去渲染变化的数据，没有变化的数据直接从缓存中取大大的减少性能开销
8、v-on:click//等同于@
9、v-model v-model.trim //实现双向绑定的 默认的只在input上
10、v-model.number //绑定的形式是number会自动帮助我们转化
11、v-model.trim// 会自动帮助我们实现收尾去除空格
12、v-model.lazy //当光标失去的时候才会去变化响应
13、v-pre// 写什么就是什么
14、v-cloak// 当vue.js还没加载之前写什么就是什么，当vue.js 加载的之后会首先解析v-cloak
15、 v-once //数据渲染只渲染一次第二次变化的时候不会在去检测渲染
