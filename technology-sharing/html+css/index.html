    # h5c3基础

    • div

    默认独占一块，由内容撑开，可以设置宽高
    常见的块元素：h1~h6、p、div、ul、table

    • span

    默认独占一行，由内容撑开 ，不能设置宽高
    块级元素可以设置宽高，而行元素不能设置宽高，即使设置了也无效。 常见的行元素：span、a、input、select等

    • 嵌套关系

    块标签可以嵌套块标签和行标签，但是p标签不能嵌套块标签; 行标签只能嵌套行标签，不能嵌套块标签，但是行标签不能嵌套a标签.

    • boder-style 边框样式

    one：无边框;
    solid：实线；
    dotted：点状边框；
    dashed：虚线

    # 盒模型的组成

    • margin：元素的外边距
    • padding：元素的內边距
    • border：元素的边框
    • content：元素的内容

    行标签不能设置上下margin

    # a标签与4个伪类

    • target 使用target属性，可以定义被链接的文档在何处显示
    <a href="https://www.baidu.com" target="_blank">百度</a>
    1

    target属性的值为"_blank"时, 代表链接将在浏览器新窗口打开.

    • :link 匹配所有未访问的链接(没有被点击过的链接)
    • :visited 匹配已经被访问过的链接
    • :hover 设置鼠标悬停时的链接样式
    • :active 匹配激活的链接(正在被点击的链接)

    注意： :hover 必须在 :link 和 :visited 后定义才能有效
    :active 必须在 :hover 后定义才能有效.
    伪类名称不区分大小写

    • hr/ 标签在 HTML 页面中创建一条水平线。
    • br/ 可插入一个简单的换行符。
