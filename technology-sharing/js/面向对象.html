# 面向对象编程

# 面向对象与面向过程

  • 面向过程

分析出解决问题所需要的步骤，然后使用函数一步步解决，依次调用

  • 面向对象

把事务分解成各个对象，建立对象不是为了完成一个步骤，而是为了描述解决问题中的行为。

# 面向对象的三大机制

  • 封装

将同种抽象的对象的属性，方法抽象成为一个类，通过类的方法来访问类。

  • 继承

龙生龙，凤生凤

  • 多态

多种功能，多种方法。
