# 7. 深浅拷贝

  • 什么是深拷贝 、 什么是浅拷贝

浅拷贝：也就是拷贝A对象里面的数据，但是不拷贝A对象里面的子对象
深拷贝：会克隆出一个对象，数据相同，但是引用地址不同（就是拷贝A对象里面的数据，而且拷贝它里面的子对象）

    var o=dest

        if(typeof obj[key]==='Object'){// 判断是不是数组或者对象

            deep(0[key],obj[key])   //递归
        }else{
            o[key]=obj[key]

    }
